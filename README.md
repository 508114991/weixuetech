# weixuetech
培训管理系统

一、概述
+ 公司名字：江西为学技术有限公司
+ 系统名称：为学教育管理系统
+ 系统内容：公司办公系统（OA）、培训管理系统
+ 项目启动期：2016年12月份
+ 项目计划完成时间：2017年12月

二、开发相关
+ 代码语言
    + 后端：PHP - ThinkPHP5框架
    + 前端
        + 前台：BootStrap
        + 后台：ACE1.4 基于 Boostrap 的管理后台
+ 说明
    + 全公司网上办公室都用这个系统
    + 分角色登录
        + 管理员/总经理
        + 各部门经理
        + 各部门职员

三、协议
+ 代码基于ThinkPHP5.0.3和BootStrap
+ 仅为教学测试使用，不涉及商业用途
